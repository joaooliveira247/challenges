PERGUNTA 2: "Qual a diferença entre REST e SOAP?"
COMO RESPONDER: REST é leve, usa HTTP e JSON; SOAP é mais pesado, baseado em XML e protocolos rígidos.

PERGUNTA 3: "Como você garantiria a segurança de uma API?"
COMO RESPONDER: Autenticação JWT, criptografia de senhas, validação de entradas e HTTPS.

PERGUNTA 4: "O que é um banco relacional e quando usar NoSQL?"
COMO RESPONDER: Relacional organiza dados em tabelas com relações; NoSQL é útil para dados não estruturados ou alta escalabilidade.

PERGUNTA 5: "Você já trabalhou em equipe usando Git?"
COMO RESPONDER: Explique como usou branches, pull requests e versionamento em projetos colaborativos.
